Document Distance

Problem Definition

Define a document as a sequence of words (alphanumeric characters), removing all the punctuation and stuff from it all. If we have two such documents, how similar are they (how many words do they have overlapping)?

Solution

Since we don't care about the order of the words in each document D , we can think of each document as a vector, with D(w)  being the number of occurrences of the word w  in D . We can then define distance as:

d'(D1,D2)=D1⋅D2=∑WD1[W]⋅D2[W]

This does work as an okay start, but it does not scale well. This formula really counts the number of words we have in common, so despite two large documents having a high percentage of words in common, this formula would yield a very high value, because a small percentage of a large document is still a lot of words. So, we need to clamp this to a fixed range using an alternate definition of the dot product, D1⋅D2=|D1||D2|cos(θ) . Defining:

d''(D1,D2)=d'(D1,D2)|D1|⋅|D2|

We can see that for θ∈[0,π2]

θ=cos−1(d'(D1,D2)|D1|⋅|D2|) , thus clamping it to a value. If θ is zero, both documents are exactly the same, and at the other end of the range, we have fully different documents.

 

Algorithm

Document Distance:

/*

 * 1. Split each document into words

 * 2. Count word frequencies (document vectors)

 * 3. Compute dot product

 */

 

def document_distance(doc1, doc2):

     list1 = get_list_of_words(doc1)

     list2 = get_list_of_words(doc2)

 

     d1 = count_frequencies(list1)

     d2 = count_frequencies(list2)

 

     sum = dot_product(d1, d2)

 

     normalized = sum / (len(list1) * len(list2))

 

     return normalized

 

def get_list_of_words(doc):

     start = 0

     end = 1

     A = ord(‘A’)

     Z = ord(‘Z’)

     a = ord(‘a’)

     z = ord(‘z’)

     zero = ord(‘0’)

     nine = ord(‘9’)

     space = ‘ ‘

     word_list = []

 

     for c in doc:

          val = ord(c)

          if A < val < Z or a < val < z or zero < val < nine:

               end += 1

          elif val == space:

               word_list.append(doc[start:end])

               start = end

               end += 1

     return word_list

 

def count_frequencies(word_list):

     count = {}

 

     for word in word_list:

          if word in count:

               count[word] += 1

          else:

               count[word] = 1

 

     return count

 

def dot_product(d1, d2):

     sum = 0

     for key in d1:

          if key in d2:

               sum += d1[key] * d2[key]

 

     return sum
